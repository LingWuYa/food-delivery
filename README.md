# food-delivery

> 基于vue2.0构建的仿饿了么外卖app应用
  * 实现了外卖app的核心功能页面，包括：
    + 购物车功能
    + 商品展示页
    + 商品评价页
    + 商家评价页

## 项目运行

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## 项目技术栈
* MVVM框架: vue 2.0
* 前端路由: vue-router
* 数据通讯: axios
* JS滚动库: better-scroll
* 构建工具: webpack3.0
* 语法检查: eslint
* 样式预处理器: stylus
